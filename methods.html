METHODS NOTE

Survey geometry with visible measurement context.

NĀSADĪYA converts catalogue coordinates into browser-readable spatial views while preserving the limits of the original measurement. The resulting geometry is a visual navigation convention, not an inferred reconstruction.

Coordinate placement

Right ascension, declination and redshift-derived distance define observer-centred Cartesian positions. Local 2MRS views can be filtered through a finite Cartesian slab; deeper DESI views remain radial lightcones.

Browser overview

Large surveys use deterministic real-row overview samples for interactive rendering. The overview does not alter the source catalogue; its visible count and the underlying catalogue count are shown separately.

Selection effects

Footprints, masks, target selection and photometric-redshift uncertainty are treated as first-class context. Empty regions are not interpreted as empty physical space.

Provenance

Clicking a point opens the survey, source table or release, object identifier, angular coordinates, redshift field and the visual-distance note.
