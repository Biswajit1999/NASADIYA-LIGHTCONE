2MRS

LOCAL SPECTROSCOPIC ANCHOR

2MRS provides the nearby all-sky baseline used for Cartesian slices and observer-centred lightcone views. The public browser package includes the real catalogue-derived layer used by the explorer.

Use this layer to inspect nearby observed structure while keeping the Zone of Avoidance and magnitude-limited selection visible.

DESI DR1 LSS

DEEP SPECTROSCOPIC FIELD

The current local build contains 6,093,818 observed LSS rows. The public browser overview shows 125,000 deterministic real rows while the full-resolution spatial tile store remains external to Git history.

DESI is not an all-sky map. North/South separation and dark regions reflect real footprint, targeting masks and completeness cuts.
