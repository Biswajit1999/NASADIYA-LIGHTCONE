2MRS provides recession velocity. The display uses z ≈ cz / c only as a spatial navigation convention.

The local slice retains real rows inside a finite Cartesian Z-slab. It is a viewing selection, not a reconstructed filament model.

This is a browser display budget. It does not alter the underlying catalogue or source provenance.

DESI tracer classes real target classes

Tracer classes are parsed from the real DESI object identifier. Disabling a class changes the rendered view only; it does not alter the source catalogue.

Colour treatment

2MRS is magnitude-limited and does not observe the whole sky uniformly. Its Zone of Avoidance is a survey selection effect, not physical empty space.
