# Conference poster assets

- `nasadiya_lightcone_A0_white_poster.svg` — A0 portrait vector poster for print or PDF export.
- `nasadiya_lightcone_A0_dark_poster.svg` — A0 portrait vector poster with dark background.
- Matching PNG files are 2104 × 2976 previews.

The poster text reflects the current platform state: 2MRS (43,533 galaxies), DESI DR1 LSS (6,093,818 observed rows), a radial extent of approximately 6.95 Gpc for the visualised redshift window, and a look-back time of approximately 11.19 Gyr at z = 2.5.

The poster identifies the representation-fidelity analysis as in progress. It does not claim quantitative validation results that have not yet been completed.
