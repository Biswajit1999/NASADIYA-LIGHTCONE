# Banner-art direction prompt

Use the following prompt for a repository social banner or README header image. Generate the image without text, logos, labels, or watermarks; the repository title should be added as accessible HTML/Markdown rather than AI-rendered text.

```text
Wide 3:1 scientific visualisation banner for an open-source project called NĀSADĪYA LIGHTCONE. Create an elegant, research-grade deep-space data image rather than a generic sci-fi poster. Show three observational regimes in one coherent, physically restrained composition: on the left, a nearby all-sky galaxy field represented only by sparse measured cyan points with a subtle dark Zone-of-Avoidance band; in the middle, a calm transition through observer-centred distance space; on the right, two separated deep spectroscopic point-cloud footprints representing genuine North and South survey caps, with tracer populations expressed very subtly in cyan, warm amber, soft green, and violet. The composition must look like a high-end astronomical survey visualisation: black-blue background, delicate depth haze, precise point distributions, no fake filaments, no connecting web lines, no galaxies painted as spirals, no nebulae, no planets, no telescope, no grid sphere, no dashboard panels, no people. Preserve genuine-looking survey incompleteness: the dark gaps are part of the composition. High contrast but restrained, premium editorial aesthetic, realistic dimensional depth, clean enough for a GitHub social-preview banner. No text, no numbers, no logos, no watermark. 3:1 aspect ratio.
```
