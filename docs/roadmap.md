# Roadmap

- Real Table 3 download and reproducible processing.
- Observer-centred WebGL map.
- Object inspector and source fields.
- Survey caveat panel and no generated galaxy layer.

- Tile manifest generated off-repository.
- Catalogue release / target class filters.
- Spectroscopic redshift quality, tile availability, and footprint views.

## v1.0 — reproducible research states

- URL-encoded camera and filter state.
- Explicit cosmology choice.
- Per-layer data version and provenance panel.
